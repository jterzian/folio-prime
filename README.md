# Portfólio Profissional | João Terzian

Este é o meu portfólio desenvolvido como projeto final, utilizando **React** e **CSS Modules**. O projeto destaca minhas habilidades em front-end e minha trajetória em Infraestrutura e Tecnologia.

## 🔗 Demonstração Online
[Clique aqui para acessar o projeto](https://folio-prime.vercel.app/)

## 🛠️ Tecnologias e Conceitos Utilizados
- **React.js**: Hooks (useState, useParams), Componentização.
- **React Router Dom**: Navegação dinâmica e rotas de detalhes.
- **CSS Modules**: Estilização modular e design Neo-Brutalista.
- **Responsividade**: Adaptado para Desktop e Dispositivos Móveis.

## 💻 Projetos em Destaque
- **Agência Criativa**: Foco em UI/UX e performance de carregamento.
- **Pet & Style**: Lógica de e-commerce e manipulação de estados complexos.
- **React Avançado**: Gestão de tarefas com Context API e LocalStorage.

## ⚙️ Como executar localmente
1. Clone este repositório.
2. No terminal, execute `npm install` para instalar as dependências.
3. Execute `npm start` para iniciar o servidor de desenvolvimento.
